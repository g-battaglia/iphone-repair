# Error 4030

## Description
An undocumented error during iOS restore. Not listed in Apple's official support pages or The Apple Wiki (Restore Errors).

## Classification
- Family: 40xx (USB/communication errors)
- Apple documents: 4005, 4013, 4014, 4037 — but skips 4030
- Likely generated by the restore OS on the device side, not by iTunes/Finder

## Observed behavior
In Recovery mode:
1. iBEC sent successfully
2. DeviceTree, SEP, KernelCache all sent successfully
3. Tool waits for the device to disconnect and reconnect in restore mode
4. Device never enters restore mode; falls back to Recovery
5. Error "image personalization failed" or timeout

## Cause in our case
See [[root-cause]] — the iOS 15.3.1 iBoot couldn't validate iOS 26.4.1 firmware components.

## Solution
See [[dfu-vs-recovery]] — use DFU mode to bypass the local iBoot.
